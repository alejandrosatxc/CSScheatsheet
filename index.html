CSS cheatsheet

Pseudo Classes

Pseudoclasses like :hover will apply styling when an element has a mouse scrolled over it. Typically requires 2 CSS selectors, one with mouse hover and the other without.

Pseudo Elements

Two of the most common pseudo elements are ::before and ::after. Using them places a 'fake' element before or after the targetted element, can be combined with pseudo classes for fun effects.
